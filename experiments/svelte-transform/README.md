# Direct-Rolldown Svelte Transform Experiment

This directory contains the [formal result](./2026-07-11-svelte-results.md) and raw data for the pinned isolated Svelte compiler-transform case. The 24-component fixture regresses at every tested worker count, while the synthetic 1,340-component confirmation reaches a 1.36x paired median wall-time speedup at four workers with higher CPU and RSS. It is an upper-bound kernel result: one fan-out entry makes every SFC a root and SFC dependencies are externalized. The executable fixture, upstream corpus manifest, extraction rules, and MIT license live on Rolldown branch `research/parallel-js-plugin-svelte-case` at `20cbc043ccf1ab730ded962db1f413abde15753d`.
