# Direct-Rolldown Svelte Transform Experiment

This directory contains two complementary direct-Rolldown Svelte results. The [isolated compiler-kernel report](./2026-07-11-svelte-results.md) shows a 24-component negative control and a 1.36x four-worker upper bound over 1,340 real SFC sources made independent by a synthetic fan-out. The [graph-preserving registry UI report](./2026-07-11-svelte-registry-graph-results.md) follows all project-local edges from 56 real shadcn-svelte barrels: worker-4 reaches a smaller 1.117x paired median speedup, wins all 15 pairs, reduces median per-run maximum event-loop delay from 314.6 ms to 9.9 ms, and costs 2.84x user CPU and 2.17x peak RSS. Worker-8 loses. The two results separate a prepared compiler ceiling from value on a representative project subgraph.

The executable fixtures, upstream manifests, extraction rules, expected graph, and retained MIT license live on pushed Rolldown branch `research/parallel-js-plugin-svelte-case`. The isolated result pins `20cbc043ccf1ab730ded962db1f413abde15753d`; every formal graph result pins clean commit `1074399c2e3b0858388e5a7dee586388c76c82f6`. This is not a full SvelteKit application, Vite build, or official Svelte plugin result.
