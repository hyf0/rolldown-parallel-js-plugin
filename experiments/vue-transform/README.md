# Direct-Rolldown Vue Transform Experiment

This directory contains the [durable result](./2026-07-11-vue-icon-results.md) and raw data for the pinned `unplugin-vue/rolldown` transform case. The executable fixture lives on the Rolldown research branch and prepares its pinned upstream corpus outside every measured child process. The 166-SFC case is a reproducible negative result: every tested worker count regresses in the 30-round confirmation, while worker-1 still materially improves Node.js main-loop responsiveness.
